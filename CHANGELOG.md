# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-03-30

### Added
- Initial open-source release of ACP (Agent Collaboration Platform)
- Electron 28 desktop shell with React 18 + TypeScript + Vite
- 4-pane terminal grid with xterm.js for multi-agent orchestration
- Node.js + Express backend API with agent lifecycle management
- Agent collaboration engine: chat, mail integration, broadcast messaging, party formation
- Autonomous operation subsystem with supervisor and human escalation
- VibeSQL storage adapter (PostgreSQL via `POST /v1/query`)
- Kanban board and session management APIs
- Monorepo structure with `packages/api` and `packages/desktop` workspaces

### Fixed
- Clone URL corrected in README
